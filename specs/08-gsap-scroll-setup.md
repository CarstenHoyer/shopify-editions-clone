# 08-gsap-scroll-setup

> Epic: Scroll Animations
> Dependencies: 06-page-sections

---

## Goal

Configure GSAP ScrollTrigger for scroll-based animations.

## Requirements
- GSAP and ScrollTrigger registered
- useGSAP hook for React integration
- ScrollTrigger markers for debugging (toggle-able)
- Smooth scroll behavior

## E2E Test

Write test in `e2e/scroll.spec.ts`:
- Scroll to bottom and back works
- No JS errors on scroll

## Done when
- [ ] `npm run build` passes
- [ ] ScrollTrigger works (test with console.log on enter)
- [ ] Smooth scroll active
