# 16-responsive-design

> Epic: Polish & Performance
> Dependencies: 06-page-sections

---

## Goal

Make the site responsive for mobile and tablet.

## Requirements
- Mobile layout (< 768px)
- Tablet layout (768px - 1024px)
- Touch-friendly interactions
- Adjusted 3D camera for mobile
- Text sizes and spacing for small screens

## E2E Test

Write test in `e2e/responsive.spec.ts`:
- Page renders correctly at 375px width
- Page renders correctly at 768px width
- No horizontal scroll

## Done when
- [ ] `npm run build` passes
- [ ] Mobile layout works
- [ ] Touch scroll works
